### 2. رفع الملفات
```bash

git branch -M main

git push -u origin main
```

### 3. تفعيل GitHub Pages
1. اذهب إلى **Settings** في المستودع

3. في **Source** اختر `Deploy from a branch`
4. اختر الفرع `main` والمجلد `/ (root)`
5. انقر **Save**
6. انتظر دقيقتين ثم زر الرابط المُنشأ
